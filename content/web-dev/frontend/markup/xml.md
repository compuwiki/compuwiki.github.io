XML es un lenguaje de marcado que se utiliza para almacenar y transportar datos de manera estructurada. Es ampliamente utilizado en configuraciones, intercambio de datos y documentos.

## Características Principales

- Permite definir etiquetas personalizadas.
- Es legible tanto por humanos como por máquinas.
- Se usa comúnmente en APIs, bases de datos y configuración de software.

## Sintaxis Básica

Ejemplo de un documento XML:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<libro>
    <titulo>Ejemplo de XML</titulo>
    <autor>Juan Pérez</autor>
    <año>2024</año>
</libro>
```
