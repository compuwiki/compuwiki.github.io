CSS es el lenguaje utilizado para describir la apariencia y el diseño de una página web. Permite aplicar estilos a los elementos HTML y mejorar la experiencia del usuario.

## Características Principales

- Separa la estructura (HTML) del diseño.
- Permite reutilizar estilos a través de múltiples páginas.
- Compatible con animaciones y diseños responsivos.

## Sintaxis Básica

Ejemplo de código en CSS:

```css
/* Estilo para el cuerpo del documento */
body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    color: #333;
}

/* Estilo para los encabezados */
h1 {
    color: #007bff;
    text-align: center;
}
```
